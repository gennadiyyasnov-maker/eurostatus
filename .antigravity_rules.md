# Antigravity Agent Rules

CRITICAL: Do NOT use Python (`python -c`, `script.py`) to execute find-and-replace strings or manipulate HTML/CSS source code files iteratively! 

The user has explicitly forbidden script-based site patching because it litters the directory with temporary `.py` files and fails randomly due to escaping issues. 

**MANDATORY DIRECTIVE:**
Always use native Antigravity file manipulation tools: `replace_file_content`, `multi_replace_file_content`, or `write_to_file`. Treat all HTML/CSS edits as manual, surgical replacements using the tools provided.
