# КРИТИЧЕСКИЕ ПРАВИЛА ДЛЯ ИИ (AI RULES)

**ДАТА ЗАПИСИ:** 2026-04-14
**ПРОЕКТ:** EuroStatus (eu-citizenship)

**СТРОГОЕ ПРАВИЛО ОТ ПОЛЬЗОВАТЕЛЯ:** 
1. **ЗАПРЕЩЕНО ИСПОЛЬЗОВАТЬ СКРИПТЫ (PYTHON/BASH) ДЛЯ ВЕРСТКИ И ИЗМЕНЕНИЯ CSS/HTML.** 
2. Все изменения в коде (CSS, HTML, JS) должны вноситься **исключительно ВРУЧНУЮ** через специализированные инструменты поиска и замены по строкам (например, `replace_file_content` или `multi_replace_file_content`).
3. Запрещено использовать регулярные выражения в "костыльных" Python скриптах для массовой замены, так как это приводит к необратимым багам, тихим поломкам и потерянной верстке.
4. При работе с адаптацией (мобильная версия), изменения не должны затрагивать десктопную версию. Всегда тщательно проверять закрывающие скобки медиа-запросов `@media`.

Нарушение этих правил приведет к поломке проекта. ЭТОТ ФАЙЛ ОБЯЗАТЕЛЕН К ПРОЧТЕНИЮ ПЕРЕД КАЖДОЙ СЕССИЕЙ.
